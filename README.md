## Overview

This project predicts telecom customer churn using Balanced Logistic Regression and segments customers into Low, Medium, and High risk categories to support retention strategies.

## Objective

Predict customer churn

Handle class imbalance

Identify key churn drivers

Provide business recommendations

## Tools Used

Python, Pandas, NumPy, Scikit-learn, Matplotlib

## Model Performance

Accuracy: 77%

Recall (Churn): 78%

Precision: 57%

Class imbalance was handled using class_weight="balanced" to improve churn detection.

## Business Impact

The model identifies high-risk customers and provides actionable insights to reduce churn and improve customer retention.
